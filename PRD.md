# Product Requirements Document (PRD) - Flower Shop MVP

## 1. Мета проєкту (Objective)
Створення MVP вебзастосунку для онлайн-продажу квіткових композицій. Основна ціль — надати користувачу зручний інтерфейс для перегляду каталогу та базовий механізм додавання товарів до кошика.

## 2. Скоуп MVP (Scope)
Мінімальний набір функцій (MVP), який буде реалізовано:
* **Header (Шапка):** Містить логотип магазину та віджет кошика (відображає загальну кількість обраних товарів та суму).
* **Каталог товарів:** Сітка карток товарів. Мінімум 4-6 позицій (фото, назва букета, ціна). Дані зберігаються локально (mock data).
* **Функціонал кошика:** На кожній картці товару є кнопка "В кошик". При натисканні лічильник товарів та сума в шапці автоматично оновлюються (стан зберігається через хук `useState`).

## 3. Технічний стек (Tech Stack)
* **Фреймворк:** React (ініціалізація через Vite).
* **Стилізація:** CSS Modules.
* **Контроль якості коду:** ESLint, Stylelint.
* **Таск-трекер:** GitHub Issues.

## 4. Майбутні покращення (Out of Scope for MVP)
* Окрема сторінка оформлення замовлення (Checkout).
* Підключення реального бекенду та бази даних.
* Система авторизації користувачів.
